# Android APK Setup Guide - Trusted Web Activity (TWA)

This guide will help you package the Production Management App as an Android APK using Trusted Web Activity (TWA).

## Prerequisites

- Android Studio installed
- Java Development Kit (JDK) 8 or higher
- Node.js and npm/pnpm installed
- Your app deployed to a public URL (Internet Computer canister URL)
